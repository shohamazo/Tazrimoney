# **App Name**: Tazrimony

## Core Features:

- Shift Clock-In/Out: Allows users to start and end shifts, recording start and end times, showing the user the saved shifts and allowing edit of start\end date, job type.
- Automated Israeli Labor Law Calculator: When the shift end time is selected, this tool automatically calculates the correct payment by applying all relevant Israeli labor laws (night hours, weekly rest days, overtime, travel reimbursement).
- Expense Tracking: Allows users to manually input expenses, categorizing them as recurring or one-time.
- Budget Management: Displays planned vs. actual spending per category, allowing users to edit alert thresholds.
- Dashboard: Displays real-time financial metrics, including total earnings, total spent, net balance, days worked, and hours to zero balance.
- Job Management: Enables users to manage jobs, including job name and base hourly rate.
- Financial Reporting: Generates historical monthly financial reports with graphs and summary tables.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and reliability in financial management.
- Background color: Light gray (#F5F5F5), providing a neutral backdrop for readability and focus.
- Accent color: Yellow (#FFEB3B) to highlight important information and call-to-action elements, ensuring key functionalities stand out. This is chosen for a high contrast against the primary.
- Body and headline font: 'Inter' sans-serif font for a clean and modern aesthetic that supports Hebrew characters.
- Use clean, modern icons to visually represent different categories and actions, enhancing usability in the RTL layout.
- Implement a strict right-to-left layout, mirroring elements to ensure natural navigation for Hebrew-speaking users. Start with the most important CTAs on the right.
- Subtle animations to provide feedback on interactions and guide users through the application, making the interface more engaging and intuitive.
